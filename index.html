Google Drive Auto-Sync

Your data auto-saves to nutritrack-data.json in your Drive and loads on every device you sign in from. One-time setup:

1
Go to console.cloud.google.com → create a new project named NutriTrack
2
APIs & Services → Library → search Google Drive API → Enable it
3
APIs & Services → OAuth consent screen → External → fill App name → add your Gmail as a test user → Save
4
Credentials → Create Credentials → OAuth Client ID → type: Web application
Add your app's URL under Authorized JavaScript origins.
⚠ file:// won't work — host the HTML file for free on tiiny.host or GitHub Pages, then add that URL.
Copy the Client ID.
5
Paste your Client ID below and sign in. That's it — sync is automatic from then on.

NutriTrack only accesses files it creates — it cannot read any other files in your Drive.
